## About

### Mission
**Datathon 2026** is a nationwide innovation challenge by the **Karnataka State Police**, aimed at leveraging data to build smarter, safer, and more efficient systems for law enforcement and public safety.

### Synergy
Bringing together students, professionals, developers, data scientists, and analysts to tackle complex real-world challenges using **advanced data-driven approaches**.

### Objective
Participants will work on impactful problem statements to build scalable solutions and contribute to strengthening **public safety infrastructure** through technological innovation.
